# Demo Script — 3 to 5 Minutes

## 1. Opening (20s)
"This is OpenClaw AI Trader, a modern crypto trading dashboard that combines live market monitoring, GPT-based analysis, and OpenClaw-powered execution orchestration."

## 2. Show the live chart (40s)
- Switch between BTC, ETH, SOL, AVAX
- Point to live candlestick updates
- Mention Binance streaming data

Say:
"The chart is not static. We bootstrap from live REST data and continue with real-time Binance kline streaming, so the operator always sees active market structure."

## 3. Show AI reasoning (50s)
- Highlight AI signal engine
- Mention confidence, action, entry, SL, TP
- Show agent reasoning feed

Say:
"Instead of only showing a signal, GPT explains the setup in practical trading terms — what action is recommended, where to enter, where to protect downside, and how to take profit."

## 4. Show execution control (50s)
- Point to trade module
- Click execute button
- Show order queue changing state

Say:
"Once a trade plan is approved, the dashboard can hand off the execution request to an OpenClaw agent. This bridges analysis and action while keeping the workflow structured and trackable."

## 5. Show execution status (30s)
- Point to submitted / approved / executed states

Say:
"We also expose the execution lifecycle so users can see exactly where an order is in the pipeline."

## 6. Close with value (30s)
Say:
"The core value here is simple: traders need one place to monitor, understand, and act. OpenClaw AI Trader makes that possible with live data, explainable AI, and agent-driven orchestration."

## Optional Q&A bullets
- human-in-the-loop by default
- can be extended to paper trading or broker APIs
- OpenClaw enables execution workflows and automation expansion
- fallback mode keeps demo resilient even if one API is unavailable
